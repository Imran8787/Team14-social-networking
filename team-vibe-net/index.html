About the Platform

This is a creative microblogging experiment by Team 14, designed for quick status sharing.

Feel free to post short updates with a 140-character limit.

Key Features

  • Write quick posts with hashtag and mention support.
  • Stylish formatting for #tags and @users.
  • Auto-generated like and share counts.
  • Dynamic character countdown.
  • Theme toggle for light/dark modes.
  • User avatar and profile metadata.
  • Responsive on all screen sizes.
  • Expandable future options (edit, delete, reactions).
  • Organized structure using HTML, CSS, and JavaScript.
  • Post archive view and scrolling layout.
  • Theme preference saved using browser storage.

Planned Improvements

  1. Login/signup system.
  2. Save posts locally or in a backend.
  3. Include media (images/emojis).
  4. Timestamps, post editing.
  5. Auto detect dark/light theme.
  6. Admin controls for posts.
  7. Multi-language interface.
  8. Post-sharing links.
