About Team Vibe Net

This is a social microblog platform built for fun and practice by engineering students.

Each post is a quick shout with a character limit of 140 characters.

Features

  • Create simple posts with mentions and hashtags.
  • Styled display for hashtags and mentions.
  • Random retweet and like counters.
  • Live character counter.
  • Day/Night theme toggle.
  • Profile image and metadata for each post.
  • Responsive layout for mobile and desktop views.
  • Footer with copyright information.
  • Expandable features (edit, delete, like, reply coming soon).
  • Modular code split into HTML, CSS, and JS for learning purpose.
  • Post history view and scrollable feed.
  • Theme remembers user selection with local storage.

Future Enhancements

  1. Add login functionality.
  2. Store posts in local storage or database.
  3. Support images and emojis in posts.
  4. Post timestamps and edit options.
  5. Dark mode auto-detection.
  6. Admin panel for moderation.
  7. Multilingual support.
  8. Post sharing via link generation.
